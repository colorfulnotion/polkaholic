You can generate the above summary data using the following queries using the public dataset `substrate-etl` in Google BigQuery:


### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.polkadot.block2002`  where signed and LAST_DAY(date(block_time)) = "2022-04-30" group by logDT order by logDT
```

```
