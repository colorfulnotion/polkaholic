### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.polkadot.blocks2086`  where LAST_DAY(date(block_time)) = "2022-03-31" group by logDT order by logDT
```


### Signed Extrinsics

```
