### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.kusama.block2121`  where signed and LAST_DAY(date(block_time)) = "2022-10-31" group by logDT order by logDT
```


### Signed Extrinsics
```
SELECT date(block_time) as logDT, COUNT(*) numSignedExtrinsics FROM `substrate-etl.kusama.extrinsics2121`  where signed and LAST_DAY(date(block_time)) = "2022-10-31" group by logDT order by logDT
```
