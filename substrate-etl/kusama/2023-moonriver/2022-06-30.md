### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.kusama.blocks2023`  where LAST_DAY(date(block_time)) = "2022-06-30" group by logDT order by logDT
```


### Signed Extrinsics

```
