## Substrate-etl Queries:
You can generate the above summary data using the following queries using the public dataset `substrate-etl` in Google BigQuery:


### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.kusama.block2084`  where signed and LAST_DAY(date(block_time)) = "2023-02-28" group by logDT order by logDT
```


### Signed Extrinsics

```
